========================================
 DS 서비스 상태 위젯 - 설치 및 사용 안내
========================================

DS스토어 · 동아사이언스닷컴 · d라이브러리 3개 서비스의
상태를 화면 오른쪽 아래 트레이에 경광등 아이콘으로
상시 표시하는 프로그램입니다. (윈도우용)

30분 주기의 경량 모니터링 공개 데이터를 5분마다 읽는
방식이라 로그인·인증이 전혀 필요 없습니다.


[ 설치 방법 ]

1. DsStatus.exe 파일을 원하는 폴더에 저장하세요.
   (다운로드 폴더에 그대로 두는 건 비추천 - 실수로 지워요)

2. DsStatus.exe를 더블클릭해 실행하세요.

   ※ "Windows의 PC 보호" 파란 경고창이 뜰 수 있습니다.
      서명 없는 자체 제작 프로그램이라 뜨는 정상 경고입니다.
      → "추가 정보" 클릭 → "실행" 버튼 클릭
   ※ 그 외 보안 프로그램이 실행을 막으면 Claude Code에
      파일 경로를 주고 "설치해줘"라고 부탁하는 게 빠릅니다.

3. 트레이에 경광등 아이콘이 생기면 끝!
   (∧ 안에 숨어 있으면 마우스로 잡아 빼서 시계 옆에
    두면 계속 보입니다)


[ 사용 방법 ]

- 경광등 색 = 서비스 상태
  · 초록 = 3개 서비스 모두 정상
  · 주황(빛살) = 주의 (느림 · 이미지 깨짐 · 점검 지연)
  · 빨강(빛살) = 장애 발생!
  · 회색 = 데이터 조회 불가
- 마우스 오버 = 서비스별 응답시간 요약
- 왼쪽 클릭 = 상세 화면 (상태 · 응답시간 · 24시간 가동률)
- 오른쪽 클릭 = 새로고침 / 대시보드 열기 /
  로그인 시 자동 시작(추천!) / 종료
- 장애 발생·복구 순간에 알림이 한 번 뜹니다.

Q. 삭제하고 싶어요.
A. 오른쪽 클릭 → "로그인 시 자동 시작" 체크 해제 →
   종료 → DsStatus.exe 파일 삭제. 끝입니다.
